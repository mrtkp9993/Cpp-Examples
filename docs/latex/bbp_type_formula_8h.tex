\subsection{Detailed Description}
B\+PP Type formula calculation. 

The Bailey–\+Borwein–\+Plouffe formula (B\+BP formula) is a spigot algorithm which is an algorithm for computing the value of a mathematical constant such as π or e which generates output digits in some base (usually 2 or a power of 2) from left to right, with limited intermediate storage. The name comes from the sense of the word \char`\"{}spigot\char`\"{} for a tap or valve controlling the flow of a liquid.

Formula was taken from\+: \href{https://bbp.carma.newcastle.edu.au/}{\texttt{ https\+://bbp.\+carma.\+newcastle.\+edu.\+au/}}
