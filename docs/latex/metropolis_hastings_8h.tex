\subsection{Detailed Description}
Metropolis-\/\+Hastings algorithm. 

Metropolis-\/\+Hastings Algorithm is a Markov chain Monte Carlo (M\+C\+MC) method for obtaining a sequence of random samples from a probability distribution from which direct sampling is difficult. This sequence can be used to approximate the distribution or to compute an integral.
