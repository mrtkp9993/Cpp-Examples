\subsection{Detailed Description}
Lagrange Interpolation. 

In numerical analysis, Lagrange polynomials are used for polynomial interpolation. For a given set of points \{\textbackslash{}displaystyle (x\+\_\+\{j\},y\+\_\+\{j\})\} with no two x\+\_\+\{j\} values equal, the Lagrange polynomial is the polynomial of lowest degree that assumes at each value x\+\_\+\{j\} the corresponding value y\+\_\+\{j\} (i.\+e. the functions coincide at each point).
