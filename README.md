Currently I implemented following functions & algorithms:

* BBP Type Formula
* Fixed Point Iteration
* Golden Section Search
* Composite Simpson & Double Simpson
* Runge-Kutta 4th Order (RK4)
* Linear Congruent Generator
* /dev/random Generator
* Quadratic Formula
